# Student Performance Analysis using SQL

## Project Overview
This project analyzes student performance data using SQL in MySQL. 
The analysis focuses on identifying factors affecting student scores and detecting high-performing and academically at-risk students.

## Dataset
The dataset contains student information including:
- Gender
- Race/Ethnicity
- Parental Level of Education
- Lunch Type
- Test Preparation Course
- Math Score
- Reading Score
- Writing Score

## Analysis Performed
The SQL analysis includes:

- Data exploration and missing value checks
- Subject performance statistics
- Gender-based performance comparison
- Impact of test preparation course on scores
- Lunch type and parental education analysis
- Identification of top-performing students
- Ranking students using SQL window functions
- Detection of academically at-risk students
- Creation of SQL views for high-performing students

## Key SQL Concepts Used
- Aggregation Functions (AVG, COUNT, SUM)
- GROUP BY and HAVING
- Subqueries
- CASE statements
- Window Functions (RANK)
- SQL Views
- UNION operations

## Tools Used
- SQL
- MySQL

## Files in this Repository
- `StudentPerfm.sql` → SQL queries for analysis
- `StudentsPerformance.csv` → dataset used for analysis
