# Student Performance Analysis using SQL
This project analyzes student performance data using SQL in MySQL.

## Objectives
- Analyze student scores across math, reading, and writing.
- Study the impact of test preparation courses.
- Evaluate performance differences based on gender and parental education.

## Key SQL Concepts Used
- Aggregation (AVG, COUNT, SUM)
- GROUP BY and HAVING
- CASE statements
- Window Functions (RANK)
- SQL Views

## Dataset
StudentsPerformance.csv

## Tools Used
MySQL
SQL
